O pacote choRds

O pacote choRds pode ser utilizado para a obtenção de informações sobre os acordes de músicas. Junto com ele já vêm diversos bancos de dados relativos à música popular brasileira. Estes dados foram obtidos das páginas das bandas mais acessadas, por gênero, do site do CifraClub.

devtools::install_github("brunaw/choRds")
library(choRds)
# data(package = "choRds")
library(tidyverse)

genre <- choRds::genre

da <- choRds::all %>% 
  dplyr::filter(date > 1900) %>% # Mantém apenas os anos que fazem sentido
  dplyr::mutate(date = 
                  stringr::str_extract(date, pattern = "[0-9]{4,}")) %>%
  # Mantém apenas os anos das músicas
  dplyr::mutate(date = as.numeric(date), # Deixa as datas como valores numéricos
         acorde = stringr::str_extract(chord, # Extrai as partes fundamentais dos
                                 pattern = "^([A-G]#?b?)")) %>%  # acordes
  choRds::clean() %>% # Retira as acordes muito longos (possíveis textos) 
  dplyr::left_join(genre, by = "artist") # Traz os gêneros dos artistas

Com este resumo, conseguimos saber algumas coisas sobre os dados:

  • A música mais antiga da base é de 1957, e a mais nova do ano passado, 2017;
  • A música com a maior quantidade de acordes (não distintos) é Faroeste Caboclo da banda Legião Urbana, com 371 acordes;
  • Nenhuma música atingiu mais que 83 pontos em popularidade;
  • A escala mais comum é a de Dó (C), sendo o que o Sol (G) é o acorde mais comum (e ele está presente na escala de Dó).
  • Existe uma predominância de música sertaneja e MPB nesta base de dados;

Ou seja, com o passar do tempo, diferentes variações de acordes apareceram nas músicas. Outra forma de olhar para essa questão é tomar a quantidade média de acordes distintos nas músicas por ano:

da_g <- da %>% dplyr::mutate(date < 2018) %>%  # 2018 ainda não é um ano completo
  dplyr::group_by(date, genre, music, chord) %>% # Agrupamento por data +
                                          #  + acorde + musica
  dplyr::summarise(distintos = n_distinct(chord)) %>% # Mantém os acordes
                                                      # distintos/ano
  dplyr::summarise(cont = n()) %>% 
  dplyr::summarise(media = mean(cont), contagem = n()) # Conta quantos 
                                          # são os acordes distintos/ano

Apesar do que vimos anteriormente, parece que as média de acordes distintos em músicas eram maiores entre os anos 60 e 80. Após isso, temos como se fosse uma estabilização perto de \(\sim 10\) acordes diferentes/música.

Inserindo graus:

# Base com acordes das escalas maiores
dmaj <- choRds::deg_maj %>% 
  dplyr::mutate(degree = as.character(degree)) %>% 
  stats::setNames(c("degree", "C", "G", "D", "A", "E", "B", 
                    "F#", "Gb", "C#", "Db", "G#", "Ab", "D#", 
                    "Eb", "A#", "Bb", "F")) %>% 
  reshape::melt(id = "degree") %>% 
  stats::setNames(c("grau", "escala", "acorde")) %>% 
  dplyr::mutate(grau = as.character(grau), 
                escala = as.character(escala),
                acorde = as.character(acorde))



# Base com acordes das escalas menores
dmin <- choRds::deg_min %>% 
  stats::setNames(c("degree", "A", "E", "B", "F#", "C#", "G#", 
                    "Ab", "D#", "Eb", "A#", "Bb", "F", "C", 
                    "G", "D")) %>% 
  reshape::melt(id = "degree") %>% 
  stats::setNames(c("grau", "escala", "acorde")) %>% 
  dplyr::mutate(grau = as.character(grau), 
                escala = as.character(escala),
                acorde = as.character(acorde))

# Base com as escalas maiores, suas equivalentes e menores relativas
eqv <- choRds::eqv %>% 
  dplyr::mutate(key = as.character(key),
                minor.rel = as.character(minor.rel)) %>% 
  stats::setNames(c("escala", "menor.rel", "rep"))

da <- da %>% dplyr::mutate(escala = as.character(key))

# Adiciona os menores relativos das escalas presentes na base,
# e os acordes de acordo com a escala (ou seu menor relativo)
da_j <- da %>% dplyr::left_join(eqv[, c("escala", "menor.rel")], 
                                by = "escala") %>% 
  dplyr::left_join(dmaj, by = c("escala", "acorde")) %>%   
  dplyr::left_join(dmin, by = c("escala", "acorde")) %>% 
  dplyr::mutate(grau_x = as.factor(grau.x),        # Converte as colunas
                grau_y = as.factor(grau.y)) %>%    # em fator
  dplyr::mutate(deg = ifelse(       #
    grau_x == grau_y, 
    grau_x, grau_y))

library(plyr)

da_t <- da_j %>%   na.omit() %>% 
  dplyr::group_by(artist, genre, music, deg)  %>% 
  dplyr::summarise(contagem = n()) %>% 
  dplyr::mutate(total  = sum(contagem), 
                perc = round((contagem/total)*100, 2),
                deg = factor(deg, levels = as.factor(1:7)))

# Completando com os níveis dos graus faltantes
